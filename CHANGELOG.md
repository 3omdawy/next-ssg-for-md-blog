# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-17

### Added

- **RTL Support**: Native Right-to-Left language support with auto-detection.
- **Embeddable Mode**: New build target `npm run build:embeddable` for generating pure HTML fragments.
- **Series Feature**: Ability to link multiple posts into a sequential series.
- **Search**: Client-side fuzzy search using Fuse.js.
- **MDX Support**: Full support for `.mdx` files with React components.
- **Dark Mode**: Built-in theme switching with persistence.

### Changed

- Standardized project structure for public release.
- Updated documentation with comprehensive deployment and customization guides.
- Optimized bundle size and build performance.

### Fixed

- Addressed font loading issues for multilingual content.
- various minor UI polish and responsive design tweaks.

## [0.1.0] - 2026-01-01

### Added

- Initial beta release.
- Basic markdown rendering.
- Tailwind CSS setup.
