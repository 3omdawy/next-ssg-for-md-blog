# Security Policy

## Supported Versions

We currently support the following versions of the Next.js SSG Blog Framework:

| Version | Supported        |
| ------- | ---------------- |
| 1.0.x   | ✅ Supported     |
| < 1.0   | ❌ Not Supported |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please open a public issue.

Thank you for helping keep this project secure!
